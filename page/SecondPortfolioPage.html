1. FRONTEND DEVELOPER & WEB PUBLISHER 부분은 포토샵을 통해 글자를 둥글게 만든 후 사용 하였습니다 transform을 통해 우측 상단에 맞게 위치를 조정 하였고 rotate라는 @keyframes 를 만들어 1초 동안 돌게 만들었습니다.

2. 중앙에 보이는 네온사인 간판의 느낌은 flicker 라는 keyframe을 만들어 0%, 18%, 22%, 25%, 53%, 57%, 100% 일때에는 box-shadow: 0 0 .2rem #fff, 0 0 .2rem #fff, 0 0 2rem #bc13fe, 0 0 0.8rem blue, 0 0 2.8rem #bc13fe, inset 0 0 1.3rem #bc13fe 을 주어 색상을 표현하였고
20%, 24%, 55%일 때는 box-shadow를 none을 주어 마치 깜빡이는 것처럼 보이게 만들었습니다.
